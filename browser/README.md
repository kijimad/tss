# Browser Engine Simulator

WebブラウザのレンダリングエンジンをシミュレートするTypeScriptプロジェクト。

## 機能

- **HTMLパーサー**: HTMLテキストをトークン化し、DOMツリーを構築
- **CSSパーサー**: CSSルールを解析し、セレクタ・詳細度を処理
- **スタイル解決**: カスケード、詳細度、継承による計算済みスタイル
- **レイアウトエンジン**: ブロック/インラインレイアウト、ボックスモデル
- **ペイント**: ディスプレイリスト生成とCanvas描画
- **UI**: エディタ、DOMツリー、レンダーツリー、ボックスモデル可視化

## 使い方

```bash
npm install
npm run dev      # 開発サーバー起動
npm run test     # テスト実行
npm run build    # ビルド
```
