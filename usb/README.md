# USB Driver Simulator

TypeScript で実装した USB ドライバスタックのシミュレータ。ホストコントローラ (xHCI)、USB デバイス、プロトコル層、ドライババインドをブラウザ上で可視化する。

## 起動方法

```bash
cd usb
npm install
npm run dev
```

ブラウザで `http://localhost:5173` を開く。

## テスト

```bash
npm test
```

## 試せること (10 シナリオ)

### 1. エニュメレーション (デバイス列挙)
「起動＋エニュメレーション」ボタンをクリック。キーボード・マウス・フラッシュドライブの 3 デバイスが検出され、アドレス割当→ディスクリプタ取得→コンフィグ設定→ドライババインドの全工程がログに表示される。

### 2. パケットレベルの観察
速度スライダーを 150ms 程度にしてエニュメレーションを実行。SETUP / DATA0 / DATA1 / ACK の各パケットがリアルタイムで流れる様子を確認できる。

### 3. コントロール転送 (GET_DESCRIPTOR)
ログ中の `[DESC]` 行を確認。最初に 8 バイトのショートディスクリプタ (maxPacketSize0 の取得)、次に 18 バイトのフルディスクリプタ (VID/PID/製品名) が取得される 2 段階のプロセスを観察。

### 4. SET_ADDRESS の流れ
`[ADDR]` 行を確認。デフォルトアドレス 0 で通信後、ホストが一意のアドレス (1, 2, 3...) を割り当てる。右サイドバーでアドレス付きのデバイス一覧を確認。

### 5. ドライバの自動バインド
`[BIND]` 行を確認。デバイスクラスに基づいて自動的にドライバが選択される:
- HID (0x03) → `usbhid` (キーボード / マウス)
- Mass Storage (0x08) → `usb-storage`
- Audio (0x01) → `snd-usb-audio`

### 6. ホットプラグ
エニュメレーション後に「ホットプラグ (Audio)」をクリック。ポート 4 にオーディオデバイスが動的に追加され、単体エニュメレーションが実行される。Isochronous 転送のエンドポイントを持つ複合デバイス。

### 7. Bulk 転送
「Bulk転送テスト」をクリック。フラッシュドライブ (addr=3) の OUT エンドポイントに 1024 バイトを送信。maxPacketSize=512 で 2 パケットに分割され、DATA0/DATA1 のトグルビット交替が確認できる。

### 8. デバイス切断
「ポート3 切断」をクリック。フラッシュドライブが切断され、アドレステーブルから削除される。サイドバーに ✗ マークが表示される。

### 9. 速度の違い
エニュメレーション中の `[SPEED]` 行を確認。キーボード/マウスは Low Speed (1.5 Mbps)、フラッシュドライブは High Speed (480 Mbps)、それぞれの速度ネゴシエーション結果が表示される。

### 10. エラーケース
コード上で `createHostController()` にデバイスを接続せずにエニュメレーションを実行すると、全ポートが「空」と表示される。存在しないアドレスへの転送はエラーイベントが発生する (テストで確認可能)。

## アーキテクチャ

```
src/
  hw/hardware.ts      ← USBハードウェア (ホストコントローラ, ハブ, デバイス)
  usb/driver.ts        ← ドライバスタック (USBコア, デバイスドライバ)
  ui/app.ts            ← ブラウザ UI
  __tests__/usb.test.ts ← テスト (18ケース)
```

### レイヤー構成

```
┌─────────────────────────────────┐
│  UI (app.ts)                    │  イベント駆動レンダリング
├─────────────────────────────────┤
│  USB Core (driver.ts)           │  エニュメレーション / 転送制御
│  ├ HID Driver (usbhid)         │
│  ├ Storage Driver (usb-storage) │
│  └ Audio Driver (snd-usb-audio) │
├─────────────────────────────────┤
│  Host Controller (hardware.ts)  │  xHCI / ルートハブ / ポート管理
│  └ USB Devices                  │  キーボード / マウス / ストレージ / オーディオ
└─────────────────────────────────┘
```
