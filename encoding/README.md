# encoding-sim

文字コードとテキストエンコーディングのシミュレータ。

ASCII、UTF-8、UTF-16、Shift_JIS、Unicodeコードポイント、BOM、エンコード・デコードプロセスの可視化を提供する。

## 機能

- **ASCIIテーブル**: 0x00-0x7Fの完全なテーブル（制御文字名付き）
- **UTF-8コーデック**: 1-4バイトのエンコード・デコード、継続バイト、BOM検出
- **UTF-16コーデック**: BOM検出、サロゲートペア、ビッグ/リトルエンディアン
- **Shift_JISコーデック**: シングルバイト・ダブルバイト、ひらがな/カタカナ/漢字サブセット
- **Unicodeデータベース**: コードポイント情報、ブロック、カテゴリ
- **文字化けデモ**: 異なるエンコーディング間の誤変換シミュレーション
- **バイトレベル表示**: 16進ダンプ、2進ダンプ、ビット構造の可視化

## スクリプト

```bash
npm run dev        # 開発サーバー起動
npm run build      # プロダクションビルド
npm run test       # テスト実行
npm run test:watch # テストウォッチモード
```
