# ECS Simulator

Amazon ECS (Elastic Container Service) のシミュレーター。クラスター、サービス、タスク定義、タスク、コンテナインスタンス、ロードバランシング、オートスケーリング、サービスディスカバリをコードで表現します。

## 機能

- **クラスター管理**: 複数のコンテナインスタンス（EC2）のCPU/メモリリソース追跡
- **タスク定義**: コンテナ定義（イメージ、CPU、メモリ、ポートマッピング、環境変数、ヘルスチェック）
- **タスクライフサイクル**: PENDING → RUNNING → STOPPED、終了コード
- **サービス**: 希望タスク数、ローリングデプロイメント、サーキットブレーカー
- **タスク配置**: ビンパッキング（インスタンス最小化）、スプレッド（均等分散）
- **ロードバランサー**: ターゲットグループ、ヘルスチェック、リクエストルーティング
- **オートスケーリング**: CPU使用率ベースのターゲット追跡、クールダウン
- **サービスディスカバリ**: タスクの登録/解除、DNS風の名前解決
- **コンテナログ**: stdout/stderrのキャプチャ

## セットアップ

```bash
npm install
```

## コマンド

```bash
npm run dev        # 開発サーバー起動
npm run build      # ビルド
npm test           # テスト実行
npm run test:watch # テストウォッチモード
```
